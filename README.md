# Eldritch Estates - Vintage Architecture

A timeless website for a vintage architecture company specializing in gothic, Victorian, and classical restoration.

## Features

- Responsive design with a dark, gothic theme
- Sections for company information, services, portfolio, and contact
- Classic serif typography and elegant color scheme
- Optimized for GitHub Pages hosting

## Technologies Used

- HTML5
- CSS3 (inline styles for simplicity)

## Viewing the Site

This site is hosted on GitHub Pages. Visit [https://parisa.architecturegithub.io](https://parisa.architecturegithub.io) to view the live site.

## Customization

To customize the content, edit the `index.html` file. Update the company name, services, images, and contact information as needed.
